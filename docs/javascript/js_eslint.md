# ESLint常见命令（规则表）

1 禁用 ESLint：
::: tip 提示
```js
/* eslint-disable */
var a = 100;
console.log(a);  
/* eslint-enable */
```
:::

2 禁用一条规则：
::: tip 提示
```js
/*eslint-disable no-console */
var a = 100;
console.log(a);
/*eslint-enable no-console */
```
:::


3 调整规则：
::: tip 提示
```js
/* eslint no-console:0 */
var a = 100;
console.log(a);
```
:::
